# Security Policy

## Reporting a Vulnerability

If you discover a security issue in **BlackWire Media Bot**, please report it responsibly. You can contact the maintainer via email:

```

[mateo@callec.net](mateo@callec.net)

```

Include as much information as possible about the vulnerability, including steps to reproduce it.

## Supported Versions

Only the latest version of the bot is actively maintained. Older versions may not receive security updates.

## Security Updates

Security fixes will be addressed as soon as possible. Users are encouraged to update to the latest version regularly.
