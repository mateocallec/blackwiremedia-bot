# Roadmap

Future development plans and feature ideas for **BlackWire Media Bot**:

---

## Short-term (next 1-2 months)
- [x] Automated posting to Bluesky.
- [x] Improved AI prompts for generating post content.
- [x] Customizable themes and visual styles for generated images.
- [x] Docker containerization and helper scripts for easier deployment.

## Medium-term (3-6 months)
- [ ] Support for X (formerly Twitter) posting via API.
- [ ] Facebook page posting support.
- [ ] LinkedIn posting support.
- [ ] Post scheduling and queue management.

## Long-term (6+ months)
- [ ] Multi-language content generation.
- [ ] User dashboard for reviewing and approving posts.
- [ ] Analytics and tracking of post engagement.
- [ ] Integration with other vulnerability databases and feeds.
